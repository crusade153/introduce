일일 원가 및 생산성 집계 시스템

MES-ERP 연계를 통해 실시간 제조원가를 모니터링하고, 표준원가와의 차이를 수율/생산성 관점에서 자동 분석합니다.

주요 기능:
  • • BCG 모델링 기반 데이터 세그먼트
  • • 이상 징후 담당자 자동 알림
  • • 경영진 원클릭 실적 확인

실시간 재고 및 소비기한 관리

재고 현황과 소비기한을 실시간으로 추적하여 위험도를 선제적으로 알리고, 장기재고를 분류하여 손실을 최소화합니다.

주요 기능:
  • • 소비기한 스트레스 테스트 (0~365일)
  • • 판매 가능/불가능 재고 즉시 파악
  • • 위험 재고 자동 알림 서비스

AI 기반 근무시간 예측 시스템

개인별 근태 데이터를 기반으로 잔여 근무시간을 예측하여 52시간제 위반 리스크를 차단하고 인력 운영을 최적화합니다.

주요 기능:
  • • 근무시간 예측 시뮬레이션
  • • 구간별(경고/위험) 자동 알림
  • • 가용 인원 실시간 조회

실시간 생산성 분석 대시보드

인당/인시당 생산성을 실시간 추적하여 목표 대비 실적을 관리하고, 생산성 트렌드와 저하 요인을 즉시 파악합니다.

주요 기능:
  • • 목표 대비 실적 Gap 자동 분석
  • • 인원 변동 현황 실시간 트래킹
  • • 생산성 저하 요인 드릴다운

일일 실적 마감 모니터링 시스템

마감 진척률을 실시간 모니터링하고, 미비한 부분은 담당자에게 '핀셋 알림'을 보내 누락을 방지합니다.

주요 기능:

  • • 미진 내용(오더명) 자동 식별
  • • 담당자별 핀셋 알림 서비스
